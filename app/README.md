# App Modules
Modules relating to front-end views of the application.

## Files
* `@types`: Extra Typescript definition files.
* `assets`: External assets for html page (Just some extra css for now).
* `libs`: Required javascript libraries. These are not edited by me, and have their own licenses.
* `models`: Modules for common parts of the ui. Used in modules under `views`.
* `views`: Front-end modules for individula views.
* `app.ts`: Webix Jet app setup file
* `index.html`: Main front-end html page
